Aula 01 - A

Resumo da Aula

1° Aprendemos a instalar o nodeJs e a como rodar o javascript fora do browser pelo terminal usando comando.

Exemplo: 
  exemplo:
      
        node aula01.js

2° Aprendemos a comentar o código para fins didáticos, assim podemos entender nosso código caso esqueçamos de algo . O código comentado não é interpretado pelo navegador, assim podemos deixar varios comentários pelo código sem interferir no funcionamento da aplicação. Existem 2 tipos de comentários. Os comentários de uma linha e comentários de múltiplas linhas.

Exemplo: 
  exemplo:
      
        // Este é um comentário de uma linha!
        
        
        /*
          Este
          é
          um
          comentário de
          múltiplas linhas!
        */
