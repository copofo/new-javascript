Aula 05

Nesta videoaula, O professor está introduzindo os operadores relacionais em JavaScript, que são utilizados para realizar comparações entre valores e retornam verdadeiro ou falso com base na comparação feita. Ela começa lembrando aos espectadores sobre o novo curso de JavaScript e pergunta se estão gostando do conteúdo até agora.

Os operadores relacionais discutidos são:

  1. Maior (>)
  2. Maior ou igual (>=)
  3. Menor (<)
  4. Menor ou igual (<=)
  5. Igual (==) - Nota: Ela menciona que um único igual também é usado para atribuição de variáveis em JavaScript.
  6. Diferente (!=)
  7. Negação (!) - Nota: Ela usa a negação para inverter o valor lógico de uma expressão.

O professor cria variáveis de exemplo: num1 com valor 10, num2 com valor 5 e num3 com valor 10. Ela demonstra como usar os operadores relacionais com essas variáveis para realizar comparações e obter resultados verdadeiros ou falsos.

Exemplos de código: 

      
      
        let num1 = 10;
        let num2 = 5;
        let num3 = 10;
        
        console.log(num1 > num2);  // true
        console.log(num1 < num2);  // false
        console.log(num1 >= num3); // true
        console.log(num1 <= num3); // true
        console.log(num1 == num3); // true
        console.log(num1 != num3); // false
        console.log(!(num1 != num3)); // true (negação invertendo o valor)
        
        // Exemplo com negação
        console.log(!(num1 == num3)); // false (negação invertendo o valor)

A professora explica como os operadores relacionais funcionam em conjunto com variáveis e fornece exemplos práticos das operações de comparação. Ela enfatiza que esses operadores são usados amplamente para comparações em programação e que eles desempenham um papel fundamental na lógica de controle dos programas.

Ela encerra a aula convidando os espectadores a se inscreverem no canal, ativarem as notificações e seguirem as redes sociais do curso para continuar aprendendo nas próximas aulas.

Nota: Este exemplo de código e explicação foi baseado no conteúdo fornecido no curso Javascript Completo e Profissional, com Prof. Bruno, do canal CFBCursos.
