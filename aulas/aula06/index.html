Aula 06

Respondendo perguntas das aulas de 1 a 5

Nesta vídeo aula, o instrutor aborda uma sessão de perguntas e respostas relacionadas ao conteúdo anterior do curso. As perguntas foram coletadas de aulas anteriores e são respondidas de maneira prática e esclarecedora. Aqui estão algumas das perguntas discutidas, juntamente com exemplos de código quando aplicável:

  1. Diferença entre == e ===: O instrutor explica a diferença entre os operadores de igualdade == e === em JavaScript. O == verifica a igualdade de valores, enquanto o === verifica a igualdade de valores e tipos.
  2. Uso do Node.js para execução: É mencionado que o Node.js é uma ferramenta útil para executar código JavaScript fora do navegador, permitindo que você execute scripts no servidor.
  3. Utilização do prompt no navegador: O instrutor demonstra como usar o prompt para interagir com o usuário e obter entrada de dados pelo navegador.
  4. Diferença entre != e !==: A diferença entre os operadores de desigualdade != e !== também é explicada. O != verifica desigualdade de valores, enquanto o !== verifica desigualdade de valores e tipos.
  5. Identificação de dispositivos: Um exemplo de código é fornecido para mostrar como usar a propriedade navigator.userAgent para identificar se o acesso está sendo feito a partir de um dispositivo móvel ou um PC.
  6. Manipulação do DOM (Modelo de Objeto de Documento): O instrutor discute a estrutura de árvore do DOM, que representa a estrutura de elementos HTML em uma página. Ele menciona que o JavaScript é usado para manipular e interagir com elementos no DOM.
  7. Nomenclatura de variáveis: O instrutor aborda a nomenclatura de variáveis, incluindo o uso de prefixos como GE (global), L (local) e C (constante). Ele ressalta que isso é uma questão de organização pessoal e não tem impacto significativo no desempenho ou funcionalidade.
  8. Carregamento de scripts assíncronos: O instrutor explora o uso do atributo defer ao carregar scripts no HTML. Ele explica que isso permite que o JavaScript seja carregado em paralelo com o HTML, mas enfatiza que colocar scripts no final da página é uma prática recomendada.
  9. Importância de HTML e CSS: É enfatizado que ter conhecimento prévio de HTML e CSS é importante ao trabalhar com JavaScript para a web, especialmente ao manipular atributos, propriedades e estilos dos elementos no DOM.

A aula conclui com o instrutor incentivando os alunos a continuar se aprofundando nos conteúdos relacionados e assistindo a playlists de HTML e CSS para fortalecer suas habilidades antes de prosseguir com JavaScript.

Ativando o Modo Estrito:

Para ativar o modo estrito em seu código JavaScript, basta adicionar a linha "use strict"; no topo do seu arquivo ou função. Veja como fica: 

      

        "use strict";

Conclusão:

O modo estrito no JavaScript é uma ferramenta valiosa para desenvolvedores que desejam escrever código mais limpo, seguro e robusto. Ao adotar o modo estrito, você estará evitando erros comuns e seguindo práticas recomendadas de programação. Lembre-se de ativá-lo sempre que estiver desenvolvendo para garantir que o seu código seja da mais alta qualidade possível.
